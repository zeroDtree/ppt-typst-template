
## Quick start

### Install
```bash
git clone git@github.com:zeroDtree/ppt-typst-template.git
# git clone https://github.com/zeroDtree/ppt-typst-template.git
cd ppt-typst-template
```

- For Mac OS
  ```bash
  bash shell_script/install_locally.sh macos
  ```
- For Linux
  ```bash
  bash shell_script/install_locally.sh linux
  ```

### Usage

```bash
typst init @local/ls_ppt:0.1.0
```
